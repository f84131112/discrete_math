# Q3 程式說明

這個程式會從 CSV 檔案讀取 preorder 與 inorder 的資料，並輸出對應的 postorder 。

## 需要的套件與環境

- C++ 編譯器
- 使用標準 C++ 函式庫
不需要安裝其他額外套件。

## 使用方式

1. 準備一個名為 `input3.csv` 的檔案。

2. 輸入檔格式說明：
   - 第一行以 `#*` 開頭，其中 `*` 為題號
   - 第二行：preorder
   - 第三行：inorder

   範例如下：
   ```
   #1
   1,2,3,4,5,6
   3,2,4,1,5,6
   ```

3. 執行程式後，會自動產生 `output3.csv` ， `output3.csv` 中每一行為一題的 postorder 結果。


## 編譯方式

使用以下指令進行編譯：

```
g++ -std=c++17 q3.cpp -o q3
```

編譯完成後執行：

```
./q3
```
